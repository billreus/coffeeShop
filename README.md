### 小商城功能

- 首页：新品首发、人气推荐
- 分类列表、分类详情
- 积分等级优惠
- 商品详情
- 购物车
- 下单
- 订单列表、订单详情
- 地址管理

### 管理平台功能

- 用户管理：会员管理，地址管理
- 商城管理：商品管理，订单管理
- 商品管理：商品列表，商品上架
- 推广管理：积分配置
- 统计报表



### 技术选型

#### 后端技术

| 技术        | 说明         |
| ----------- | ------------ |
| Spring Boot | 容器+MVC框架 |
| MyBatis     | ORM框架      |
| RabbitMq    | 消息队列     |
| Redis       | 分布式缓存   |
| MySQL       | 数据库       |
| Caffeine    | 本地缓存     |
| JWT         | JWT登录支持  |
